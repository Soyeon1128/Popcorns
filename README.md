# Popcorns

## 서비스 소개
- 영화 인기 순위 및 정보 검색 서비스

## MVP (Minimal Viable Product)

  #### 1. User
  - Sign In
  - Sign Up
  - Bookmark

  #### 2. Rating (순위)
  - Popular Rating

  #### 3. Search (검색) & Information (정보)
  - Title
  - Poster Image
  - Release Date
  - Genres
  - Overview
  - Tagline
  - Actor/Actress
  - Director
  - Production Companies
  - Production Countries

## 사용 기술
- Vue.js
- Vuex
- Vue-router
- axios
- webpack
- Firebase

## 영화 정보 API
- [THE MOVIE DB](www.themoviedb.org)
